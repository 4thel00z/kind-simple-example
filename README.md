# kind-simple-example

## Motivation

I need a place to lookup smart [kind](https://kind.sigs.k8s.io/docs/) templates.

## Prerequesites

Have [kind](https://kind.sigs.k8s.io/docs/) installed:

```
GO111MODULE="on" go get sigs.k8s.io/kind@v0.11.1
```
